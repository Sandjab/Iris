# Quarantine Secret — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Permettre de désactiver temporairement la substitution d'un secret (quarantine), persisté comme propriété du secret en Keychain, piloté depuis le CLI et l'app menu-bar.

**Architecture:** Le flag `quarantined: Bool` (défaut `false`) vit dans le blob JSON `StoredMetadata` du record Keychain. `ExfilRuleEngine.evaluate` (qui fetche déjà les métadonnées par hit) retire les secrets en quarantaine du jeu de hits → inertes (ni substitués, ni scorés par R1-R5). Un RPC idempotent `secret.set_quarantined` pilote l'état ; CLI et app le consomment via les patterns 6.2 existants.

**Tech Stack:** Swift 5.9, SwiftPM, XCTest, swift-nio (déjà en place), SwiftUI/AppKit (app). Aucune nouvelle dépendance.

**Spec source:** `docs/superpowers/specs/2026-05-31-phase-6.2.x-quarantine-secret-design.md`

**Branche:** `feat/phase-6.2.x-quarantine` (déjà créée ; rename `Secrets/`→`SecretStore/` + spec déjà commitées).

```swift
public struct Secret: Codable, Sendable, Hashable {
    public let name: String
    public let allowedHosts: [String]
    public let createdAt: Date
    public let lastUsedAt: Date?
    public let usageCount: UInt64
    public let quarantined: Bool

    enum CodingKeys: String, CodingKey {
        case name
        case allowedHosts = "allowed_hosts"
        case createdAt = "created_at"
        case lastUsedAt = "last_used_at"
        case usageCount = "usage_count"
        case quarantined
    }

    public init(
        name: String,
        allowedHosts: [String],
        createdAt: Date,
        lastUsedAt: Date? = nil,
        usageCount: UInt64 = 0,
        quarantined: Bool = false
    ) {
        self.name = name
        self.allowedHosts = allowedHosts
        self.createdAt = createdAt
        self.lastUsedAt = lastUsedAt
        self.usageCount = usageCount
        self.quarantined = quarantined
    }

    public init(from decoder: any Decoder) throws {
        let c = try decoder.container(keyedBy: CodingKeys.self)
        self.name = try c.decode(String.self, forKey: .name)
        self.allowedHosts = try c.decode([String].self, forKey: .allowedHosts)
        self.createdAt = try c.decode(Date.self, forKey: .createdAt)
        self.lastUsedAt = try c.decodeIfPresent(Date.self, forKey: .lastUsedAt)
        self.usageCount = try c.decode(UInt64.self, forKey: .usageCount)
        self.quarantined = try c.decodeIfPresent(Bool.self, forKey: .quarantined) ?? false
    }
}
```
