Iris est installé

Une dernière étape pour commencer à protéger vos credentials :

  1. Ouvrez Iris depuis la barre de menus.
  2. Laissez-le générer la CA locale et l'ajouter à votre trust store système (mot de passe administrateur demandé une fois).
  3. Pour router votre terminal via Iris, lancez iris shell install dans un terminal (il affiche les lignes et demande votre confirmation avant d'écrire), ou utilisez le bouton Configurer… dans l'onglet Settings → Terminal de l'app.

Ouvrez une nouvelle fenêtre de terminal ensuite pour charger les nouvelles variables d'environnement.

Iris démarre automatiquement après l'installation et se relance à chaque connexion.
