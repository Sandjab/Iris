Avant d'installer

Prérequis

  • macOS 13 (Ventura) ou ultérieur.
  • Droits administrateur — pour installer l'app et ajouter la CA locale d'Iris au trust store système.

Comment fonctionne l'interception

Iris n'intercepte que les processus lancés depuis un shell (terminaux, scripts). L'app configure pour vous deux variables d'environnement au premier lancement :

  • HTTPS_PROXY=http://127.0.0.1:8888
  • NODE_EXTRA_CA_CERTS=~/Library/Application Support/iris/ca.pem

Les apps GUI lancées depuis le Finder, le Dock ou Spotlight (Safari, Mail, Slack…) ne sont pas interceptées — c'est volontaire : le cas d'usage cible est l'outillage CLI agentique. Le proxy n'écoute que sur 127.0.0.1, rien n'est exposé au réseau.
