Before you install

Requirements

  • macOS 13 (Ventura) or later.
  • Administrator rights — to install the app and to add Iris's local CA to the system trust store.

How interception works

Iris only intercepts processes launched from a shell (terminals, scripts). Iris can configure your terminal with your consent — either by running iris shell install in a terminal (it shows the lines and asks before writing) or by using the Configure… button in the app's Settings → Terminal. The following variables are written to your shell profile:

  • HTTPS_PROXY=http://127.0.0.1:8888
  • NODE_EXTRA_CA_CERTS="$HOME/Library/Application Support/iris/ca.pem"

GUI apps launched from Finder, Dock or Spotlight (Safari, Mail, Slack…) are not intercepted — this is by design: the target use case is command-line agent tooling. The proxy listens only on 127.0.0.1, so nothing is exposed to the network.
