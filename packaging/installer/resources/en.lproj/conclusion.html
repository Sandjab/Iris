Iris is installed

One more step to start protecting your credentials:

  1. Open Iris from the menu bar.
  2. Let it generate the local CA and add it to your system trust store (you'll be prompted for your administrator password once).
  3. It configures your shell automatically. Alternatively, run iris ca install in a terminal.

Open a new terminal window afterwards so the new environment variables are loaded.

Note: Iris does not start automatically after installation.
